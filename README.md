# EMs(Employee Management System)
An Employee Management System web application developed using Java, Spring MVC, Servlet, JSP, JDBC, and MySQL.
This project allows users to Add, Update, Delete, Find, and View All Employee records through a simple web interface.
-------------------------------------------------------------------------------------------------------------------
Technologies Used
Java (JDK 8 / 11 / 17),
Spring MVC,
Servlets,
JSP & JSTL,
JDBC,
MySQL Database,
Apache Tomcat 9,
Eclipse IDE,
HTML & CSS.
-----------------
Project Features
Add new employee
Update existing employee
Delete employee by ID
Find employee by ID
View all employees
MVC architecture
Database connectivity using JDBC Template
---------------------
Database Name
employee_db
---------------------
Table Structure
CREATE TABLE employee (
    empNo INT PRIMARY KEY,
    empName VARCHAR(50),
    empSal DOUBLE
);
